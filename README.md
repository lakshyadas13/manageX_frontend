# ManageX Frontend 

A modern task management frontend built with React and Tailwind CSS.

##  Features
- Clean and modern UI (Notion-inspired)
- Add, edit, delete tasks
- Priority, tags, notes support
- JWT-based authentication (login/register)
- Protected routes
- Responsive design
- 21st.dev inspired UI components

##  Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript / TypeScript
- Axios / Fetch API

##  Project Structure
src/
components/
pages/
services/

##  Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/managex-frontend.git
cd managex-frontend

2.	Install dependencies:
```bash
npm install

3.	Create .env file:
```bash
VITE_API_URL=http://localhost:5000

4. Run the app:
```bash
npm run dev
